

I) pip install -r requirement.txt
        Si il y a, ne serait ce qu'une seule erreur lors de l'installation. Installez les modules manuellement:

        pip install pillow
        pip install pygame
        pip install tk

        *meow*

II) Lancer le fichier serverbuild.py
        Bouton "Demmarer l'animation " -> C'est pour l'animation du gif
        Bouton "Meow Meow " -> C'est pour demmarer le serveur
        ( A noter qu'apres le clic sur bouton "Meow Meow" vous ne pourez plus faire marche arriere, sauf si vous fermez la page)
        Il est donc primordial de lancer l'animation du gif en premier! *meow*

III) Lancer le fichier clientbuild.py (2fois pour avoir une discussion entre client)
        Entrez votre pseudo dans l'encadré se situant en dessous de "Entrez votre non d'utilisateur"
        Validez votre choix en appuyant sur le bouton vert " Définir le nom d'utilisateur"
        Faire la meme manipulation sur le 2eme client! *meow*

IIII) Afin d'etablir la connexion client vers le serveur, il faut envoyer un quelconque caractere
        dans la zone d'insertion de votre message, un espace par exemple.
        Faire la meme manipulation sur le 2eme client! *meow*

IIIII) Vous pouvez commencer a discuter, Amusez vous bien ! *meow*
